## Export Flow (High Level)
1. Client serializes resume data
2. POST → `/api/export-pdf`
3. Server launches headless Chromium
4. Renders `/print` with injected data
5. Streams back A4 PDF

## Security Layers
- In-memory rate limiting (login + export)
- Security headers via middleware (CSP, frame, referrer, COOP/COEP, HSTS prod)
- Ephemeral session model (no stored user record)
